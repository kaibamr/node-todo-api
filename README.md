# node-todo-api

Todo API 

- server: node & express
- database: mongoDB & mongoose
- authentication & safety: jsonwebtoken & bcryptjs

## Installation

```
 npm install
```

## API Routes

- create new user account POST url/users with body:
``` 
{
  "email": "someEmail@email.com",
  "password": "somePassword"
}
```
